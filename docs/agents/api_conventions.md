# API Conventions

- Aktualny produkt to landing page typu waitlist-first; brak aktywnego backendu produkcyjnego.
- Nie dodawaj endpointów ani integracji bez jawnego wymagania produktowego.
- Jeśli pojawi się potrzeba API, zachowaj minimalny kontrakt i spójne nazewnictwo zasobów.
- Sekrety i dane wrażliwe muszą pozostać poza repozytorium.
