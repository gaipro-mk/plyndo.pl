# MCP and Skills Policy

- W tym repo domyślnie wyłączone są pluginy i zewnętrzne MCP serwery.
- Pozostaw `mcp_servers` wyłączone, dopóki nie ma twardego wymagania integracyjnego.
- Repo nie utrzymuje lokalnych custom skills; używaj tylko natywnych narzędzi CLI oraz lokalnej dokumentacji `docs/*`.
- Każde włączenie MCP/plugins powinno być jawne, odwracalne i uzasadnione zakresem projektu.
